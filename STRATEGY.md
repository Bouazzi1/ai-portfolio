## 7. Pricing Framework

### The Pricing Ladder

```
$25-$75    → Loss leaders (Phase 1 only, to build reviews)
$75-$200   → Standard services (bread and butter)
$200-$500  → Premium services (AI-powered, multi-feature)
$500-$1,500 → Packages (bundled services)
$1,500-$5,000 → Custom solutions (enterprise, complex)
$5,000+    → Retainer + ongoing partnership
```

### Pricing Rules
1. **Never lower prices after Phase 1** — only increase
2. **Always offer 3 tiers** (anchoring effect makes middle tier sell best)
3. **Price based on value, not time** — "This chatbot will handle 500 customer questions/month" > "This took me 3 hours"
4. **Add urgency**: "Rush delivery available for +50%"
5. **Upsell always**: Every delivery includes a suggestion for what else you can build

### How to Quote Custom Projects
1. Understand the client's problem and desired outcome
2. Estimate effort (using your templates, most things take 1-4 hours)
3. Price = (effort × your hourly rate) × value multiplier
4. Your hourly rate evolution: $25/hr → $50/hr → $100/hr → $200/hr
5. Value multiplier: 1.5x-3x based on how much money/time the solution saves the client

## 10. Delivery & Fulfillment Playbook

### Standard Chatbot Delivery (10-30 minutes)
1. Client fills out a brief questionnaire (business name, services, FAQ, colors)
2. Use chatbot template → update `config.js` with client data
3. Test the chatbot with 10+ sample conversations
4. Deliver: `config.js` + `chatbot-engine.js` + `embed.html`
5. Provide installation instructions (copy-paste the embed code)
6. Offer 7-day support window

### AI-Powered Chatbot Delivery (1-3 hours)
1. Client provides: business info, documents, FAQ, tone preferences
2. Set up OpenAI API integration with client's knowledge base
3. Configure system prompt, response limits, and safety filters
4. Test with 20+ sample conversations across edge cases
5. Deploy and provide API key management instructions
6. Offer ongoing training/maintenance retainer

### Automation Delivery (30-60 minutes)
1. Client describes their current workflow and pain points
2. Import relevant n8n/Zapier template
3. Customize with client's specific tools and credentials
4. Test end-to-end with sample data
5. Deliver with documentation and video walkthrough
6. Offer 14-day support window

### Quality Checklist (Every Delivery)
- [ ] All features working as described in the order
- [ ] Mobile responsive (if applicable)
- [ ] Error handling for edge cases
- [ ] Clean, documented code
- [ ] Setup/installation instructions included
- [ ] Follow-up message sent 3 days after delivery
- [ ] Upsell suggestion included with delivery

### Follow-Up System
- **Day 0**: Deliver + thank the client
- **Day 3**: Check in — "Everything working well?"
- **Day 7**: Ask for review — "Would love your honest feedback!"
- **Day 14**: Upsell — "Here's something else that could help your business..."
- **Day 30**: Re-engage — "Any new automation needs?"

## 14. Risk Management

### Common Risks & Mitigation

| Risk | Impact | Mitigation |
|------|--------|------------|
| Bad review on Fiverr | Destroys ranking | Over-deliver, communicate proactively, offer revisions |
| Fiverr account suspension | Lose all income | Diversify platforms early (Upwork, direct clients) |
| Scope creep | Unpaid work | Clear deliverables in order, charge for extras |
| Client ghosting | Lost time | 50% upfront for direct clients, use platform escrow |
| Burnout | Business collapses | Hire help early, use templates, take weekends off |
| Tech changes (AI evolves fast) | Skills become outdated | Continuously learn, upgrade services, follow AI news |
| Copycat competitors | Price pressure | Stay ahead with quality, reviews, and relationships |
| Platform fee increases | Lower margins | Build direct client pipeline |

### Financial Rules
1. **Save 30% of all revenue** for taxes (adjust to your country's tax laws)
2. **Reinvest 20%** back into the business (tools, hiring, ads)
3. **Keep 3 months of expenses** as emergency fund
4. **Never depend on a single platform** for more than 50% of revenue
5. **Track every expense** from day one
